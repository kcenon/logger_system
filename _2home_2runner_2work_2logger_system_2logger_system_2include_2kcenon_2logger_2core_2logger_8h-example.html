Add a writer to output logs.

Add a writer to output logs

Parameters
writerUnique pointer to the writer to add
Returns
common::VoidResult Success or error code

Adds a new output destination for log messages. Multiple writers can be added to send logs to different destinations simultaneously. Ownership of the writer is transferred to the logger.

Note
Writers are processed in the order they were added.
auto result = logger.add_writer(std::make_unique<file_writer>("app.log"));
if (result.is_err()) {
std::cerr << "Failed to add writer: " << result.error().message << std::endl;
}
