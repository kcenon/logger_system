독립 실행 모드 vs 통합 모드

독립 실행 모드 (v3.0 기본값):

// thread_system 의존성 없음
auto logger = kcenon::logger::logger_builder()
.with_standalone_backend() // std::jthread 사용
.add_writer("console", std::make_unique<console_writer>())
.build();
