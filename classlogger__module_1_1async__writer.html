Detailed Description

Asynchronous wrapper for log writers.

This class wraps any base_writer implementation and provides asynchronous writing capabilities using a background thread.

Parameters

ANSI escape sequence for color, or empty string if color is disabled

Returns appropriate ANSI color codes for terminal output:

  • trace -> Gray/Dim
  • debug -> Cyan
  • info -> Green
  • warning -> Yellow
  • error -> Red
Note
Returns empty string if use_color() is false.
Warning
