Detailed Description

Compatibility wrapper for Logger System v1.x API.

This header provides backward compatibility for applications migrating from Logger System v1.x to v2.x. All functions and macros in this file are deprecated and should be replaced with their v2.x equivalents.

Deprecated:
This entire file is deprecated. Use v2.x API directly.
