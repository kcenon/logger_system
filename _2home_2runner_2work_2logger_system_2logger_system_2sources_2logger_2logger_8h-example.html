Log a message with source location

Parameters
levelSeverity level of the message
messageThe message to log
fileSource file name (typically FILE)
lineLine number in source file (typically LINE)
functionFunction name (typically FUNCTION)

Logs a message with complete source location information for debugging. This overload is useful for tracking the exact origin of log messages.

logger.log(log_level::error, "Database connection failed",
__FILE__, __LINE__, __FUNCTION__);
